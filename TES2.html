2. Tata Cara Membuat Website Sederhana

  1. Buka teks editor seperti VS Code atau Notepad++.
  2. Buat file index.html, lalu isi dengan struktur HTML dasar.
  3. Tambahkan file CSS atau gunakan Tailwind via CDN.
  4. Tambahkan JavaScript jika perlu.
  5. Buka file HTML di browser untuk melihat hasilnya.
  6. Edit dan kembangkan sesuai kebutuhanmu!
